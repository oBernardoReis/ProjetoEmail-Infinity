PROJETO: Sistema de Disparo de Email Marketing
ALUNO: Bernardo Reis 
CURSO: Programação Full Stack 

DESCRIÇÃO:
Este projeto consiste em um sistema de disparo de email marketing desenvolvido
para fins acadêmicos. O sistema permite o gerenciamento de contatos, simulação
de envio de campanhas de email e geração de relatórios, atendendo aos requisitos
propostos.

TECNOLOGIAS UTILIZADAS:
- Backend: Python (Flask)
- Frontend: HTML, CSS e JavaScript
- Banco de Dados: SQLite

FUNCIONALIDADES:
- Cadastro, listagem e remoção de emails
- Simulação de disparo de email marketing
- Registro de data e hora dos envios
- Geração de relatório de envios
- Exportação de relatório em formato CSV
- Interface web simples e intuitiva

INSTRUÇÕES PARA EXECUÇÃO:
1. Abra o projeto no VS Code
2. No terminal, ative o ambiente virtual
3. Execute o backend com o comando:
   python backend/app.py
4. Abra o arquivo frontend/index.html no navegador
5. Utilize o sistema normalmente

OBSERVAÇÃO:
O envio de emails é simulado para fins de estudo, não realizando disparos reais.
